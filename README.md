**Full-stack Retail Management Dashboard** with real-time sales tracking, P&L analytics, and multi-user authentication.

## ✨ Features

- 🔐 **Secure Login** (Admin/Manager/Viewer roles)
- 💰 **Live KPIs** (Revenue, Profit, Orders)
- ➕ **Real-time Sales Entry**
- 📊 **Interactive Charts** (Store-wise revenue, Product profit)
- 📈 **P&L Export** (CSV reports)
- 📱 **Responsive Web App**
